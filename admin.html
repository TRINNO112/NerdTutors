📋 Excel Format Guide - Text Questions

Column Name Required Description Example
type Optional Question type: "text" or "mcq" text
text Yes * The question text Explain the law of demand
category Yes * Question category Microeconomics
marks Yes (for text) Marks for the question (1-100) 10
modelAnswer Yes (for text) The ideal answer The law of demand states...
difficulty Optional Easy, Medium, or Hard Medium
status Optional active or inactive active

📋 Excel Format Guide - MCQ Questions

Column Name Required Description Example
type Yes * Must be "mcq" for MCQ questions mcq
text Yes * The question text Which is NOT a component of GDP?
category Yes * Question category Macroeconomics
optionA Yes * First answer option Consumption
optionB Yes * Second answer option Investment
optionC Yes * Third answer option Imports
optionD Yes * Fourth answer option Government Spending
correctAnswer Yes * Correct option: A, B, C, or D C
difficulty Optional Easy, Medium, or Hard Easy
status Optional active or inactive active

Note: MCQ marks are automatically set to 1. Leave marks and modelAnswer empty for MCQ questions.

Valid Categories: Microeconomics, Macroeconomics, International Economics, Development Economics, Public Economics, Monetary Economics, Labor Economics, Environmental Economics, Econometrics, Economic Theory
